This package provides a demo MATLAB code of a two-scale image fusion method used in the following paper:

"X. Sun, Y. Zhu and X. Fu, "RGB and Optimal Waveband Image Fusion for Real-Time Underwater Clear Image Acquisition," IEEE Transactions on Instrumentation and Measurement, vol. 72, pp. 1-17, 2023."

The code and data are only free for research purposes. Please refer to the above publication if you use this code.
